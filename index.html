📄 Vercel Proxy Code

Deploy these two files to a Vercel project (free). Then paste the Vercel URL above. This solves CORS and keeps your API keys server-side.

api/proxy.js 
export default async function handler(req, res) {
  res.setHeader('Access-Control-Allow-Origin', '*');
  res.setHeader('Access-Control-Allow-Methods', 'GET,POST,OPTIONS');
  res.setHeader('Access-Control-Allow-Headers', 'Content-Type');
  if (req.method === 'OPTIONS') { res.status(200).end(); return; }
  const { target, endpoint, apikey, key } = req.query;
  try {
    let url, opts = { headers: { 'Content-Type': 'application/json' } };
    if (target === 'cricket') {
      url = `https://api.cricapi.com/v1/${endpoint}?apikey=${apikey}&offset=0`;
      opts.method = 'GET';
    } else if (target === 'gemini') {
      url = `https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent?key=${key}`;
      opts.method = 'POST';
      opts.body = JSON.stringify(req.body);
    }
    const r = await fetch(url, opts);
    const d = await r.json();
    res.status(200).json(d);
  } catch (e) { res.status(500).json({ error: e.message }); }
}

🔮 Prediction Engine — How It Works

Score Projection

predictScore(runs, overs, venueAvg) → ((runs/overs)*20 + venueAvg) / 2

Blends current run rate projected over 20 overs with venue historical average. Gives a balanced estimate as the innings progresses.

Powerplay Projection

predictPowerplay(runs, overs) → (runs/overs) * 6

Simple linear projection within the powerplay window (overs 1–6). Used to benchmark the PP phase against team and venue PP averages.

Win Probability

winProbability(runs, overs, wkts, venueAvg, isChasing, target) → base% ± wicket penalties ± venue bias

Batting-first: tracks projected vs venue average with wicket deductions. Chasing: factors required rate vs current rate, balls remaining, wickets in hand, and venue chase bias.

Par Score (DLS-style curve)

parScore(overs, venueAvg) → venueAvg * curve[Math.floor(overs)]

Non-linear curve derived from typical T20 scoring patterns. Powerplay ~28%, overs 7-15 ~52%, death ~20%. Gives a meaningful benchmark at every stage.
